Keyboard shortcuts GAMERMODE ACTIVE

Jobs View
esc Close all open panels
f Enter a quick filter
ctrlshiftf Clear the quick filter
← or w Select previous job
→ or s Select next job
k or p a Select previous unclassified failure
j or n d Select next unclassified failure
t Select next info tab
l or q Open the logviewer for the selected job
r Retrigger selected job
u or e Show only unclassified failures
i or v Toggle pending and running jobs
ctrl or cmd Toggle pinning a job during click selection
Pinboard
spacebar Add a selected job to the pinboard
b Add a selected job to the pinboard + enter related bug
c Add a selected job to the pinboard + enter classification
ctrlenter or ctrlspace Save pinboard classification and related bugs
ctrlshiftu Clear the pinboard
ctrlbackspace Delete job classification and related bugs
Autoclassify Panel
a z Ignore unclassified failure lines
s x Save all autoclassifications

? Display onscreen keyboard shortcuts
